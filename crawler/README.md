# crawler 폴더 안내

레고 제품 데이터를 수집하기 위한 **크롤링 코드**를 모아둔 폴더입니다.

## 폴더 구성

- `final/`
  - 현재 사용 중인 최종 크롤러 코드
  - 재실행 시 동일한 데이터를 얻을 수 있도록 정리된 버전

- `archive/`
  - 크롬 드라이버, RSelenium, chromote 등을 테스트하며 작성한 코드
  - 속도/에러 문제를 해결하는 과정에서 사용된 이전 버전 코드

보고서에는 `final/` 기준으로 설명하되,  
DRIP의 시행착오 기록을 위해 `archive/`의 내용도 함께 보관합니다.
