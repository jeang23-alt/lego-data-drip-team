# analysis 폴더 안내

이 폴더는 레고 데이터 DRIP 프로젝트의 **분석 코드**를 모아둔 곳입니다.

## 폴더 구성

- `final/`
  - DRIP 분석 최종본 Rmd, R 스크립트
  - 제출용 시각화 코드 및 그래프 생성 코드

- `archive/`
  - 탐색 과정에서 사용한 코드
  - 오류가 났던 버전, 실험용 시각화, 아이디어 테스트 코드 등
  - DRIP의 **R(Reasoning) 단계에서의 시행착오**를 보여주기 위함

최종 결과를 재현하려면 `final/` 폴더의 파일만 실행하면 되며,  
분석 과정과 시행착오는 `archive/` 폴더에서 확인할 수 있습니다.
